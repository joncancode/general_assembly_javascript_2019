Conditional Statements

Conditional statements are blocks of code that only run when a certain criteria is met

JavaScript supports two conditional statements: if...else and switch

If...

          if (1 > 0) {
              console.log("This is a true statement");
          }
  • This would run the statement because 1 is greater than 0

    • If... else

                if (0 > 1) {
                console.log("This is a true statement");
            } 
            else {
                console.log("The first statement did not run");
            }
  • Since the first condition isn't met, we skip it and go to the 'else'
  • Notice how the brackets line up so that it's easier to read

    • If... else if... else

                  if (name === 'Harry') {
                  console.log("You're a wizard");
              } 
              else if (name === 'Hedwig') {
                  console.log("You're an owl");
              }
              else {
                console.log("Why are you in Hogwarts?");
              }
  • We can have as many conditionals as we want
  • Notice the triple equal signs (we'll touch that later. Just know to use three)

    • Ternary Operators

  • Shorthand 'if' statements
  • let age = 12;
  • let allowed = (age > 18) ? "yes" : "no";
  • allowed would result in "no"
  • (condition) ? ___ : ___

    • Comparison Operators

    Greater than, less than

  • 1 < 3 is true
  • "b" > "a" is true
  • 12 > "12" is false
  • 12 >= "12" is true

    • Equal signs

  • Single = is used when assigning a value like we've done in the past
  • Double == is used as a check, but it does not check type
  • Triple === checks type

  • So basically
    5 == '5' is true
    5 === '5' is false

    • 1's and 0's

  • 1 == true is true
  • 0 == false is true
  • 1 === true is false, however

    • What About Objs and Arrays?

  • {} === {} is false
  • [1, 2] === [1, 2] is false
  • These are not primitive types like strings and numbers. They are more complex and are passed by reference
  • This means we cannot compare as we would value types like numbers, booleans, etc

    • Exclamations are 'Not'

  • 1 !== 2 reads '1 is not equal to 2'
  • If you want to verify the 'truthyness' or 'falseyness' of a value, you can add ! in front of a value. the returned value will be the inverse of the value in a Boolean. So if you add two ! then you'll get the Boolean value of the original one
  • !!1 is true
  • !![] is true
  • !!{} is true
  • !!null is false
  • !!"" is false

    • Boolean Logic

    There are two Binary Operators that can make more complex boolean logic

  • if (age > 12 && age < 20) : the && denotes AND
  • if (day === "Monday" || day === "Wednesday") : the || denotes OR

  • !(true) : we can still use exclamation points here as negation

    • Switch Statements

    Switch statements are like If statements, but may be more aesthetically appealing

  • The first 'case' is basically saying if (food === 'pear')
  • Why wouldn't the 'age' conditional practice work with a switch statement?

    • Fall Through Method

                        let grade = 'C';

                    switch (grade) {
                      case 'A':
                      case 'B':
                      case 'C':
                        console.log('You passed!')
                        break
                      case 'D':
                      case 'F':
                        console.log('You failed')
                        break
                      default:
                        console.log('Unexpected grade value')
                    }

    While and Do-While Loops

    While is a loop statement that will run while a condition is true

          while (true) {
        // an infinite loop!
      }
  • Be careful about conditions in while loops. Infinite loops will break your code

    • A way to counter this is with a do-while loop

          let input = 0;
      do {
          console.log(input++);
      } while (input < 10)

    For and ForEach

    We touched the for and forEach loops already, but here's a refresher

    The for loop

      let a = [1, 2, 3, 4, 5];
  for (var i = 0; i < a.length; i++) {
      console.log(i);
  }
  • good for looping through arrays
  • more useful/common than while loops

    • The forEach loop (ES6)

      ["sunny", "rainy", "windy"].forEach(function(currentValue, index, array) {
    console.log("The current weather is ", currentValue);
 });
  • currentValue can also be array[index]
