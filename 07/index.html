The DOM

  • First, let's understand how browsers work
  • Basically, the DOM is a large object that describes the structure of our content
  • Since it's an object, we can use normal techniques to get and set data
  • JS specifies some built-in methods that make using the DOM easier (e.g. Document.getElementById

  • Inline JavaScript (least desirable)

    • Different ways to manipulate the DOM (3)

  • There's a lot to take in here but these are the steps:
    1. We first create the new H1 element through the document.createElement method
    2. We create the text through the createTextNode method
    3. The text is added to the newly created H1 element

    Homework

  • Remember when we did 99 Bottles of Beer? We'll now put the lyrics on the browser
  • You'll need to create both an .html file and a .js file (or add a script inside your .html file)
  • In the html file, you'll want to have an unordered list (ul) that contains all of your lyrics
  • Each line of the song should appear inside of a list item (li) within that unordered list
  • Your JavaScript file should programatically append each line of the song to the page--no hard-coding lyrics in HTML
  • Remember for loops... They may come in handy here
