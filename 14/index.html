    • What the CRUD?

  • Back in a previous class, we looked at what goes on in the Network tab and saw examples of different requests
  • In addition to GET, there are a few other key requests to remember
  • These are commonly referred to collectively as CRUD - or the RESTful architecture
    1. Create (POST)
    2. Read (GET)
    3. Update (PUT/PATCH)
    4. Delete (DELETE)
  • To help understand how this will work, let's remember the Star Wars api
  • To create a server with Node, it's as easy as the below
    • 
const http = require('http');

const server = http.createServer(function(req, res) {
  if (req.url === '/'){
    res.write('this is the route')
    res.end();
  }
  if (req.url === '/users'){
    res.write(JSON.stringify([1, 2, 3]))
    res.end();
  }
})

server.listen(3000)

console.log('listening on port 3000')
  • However, this code is already a bit crowded and may be tough to add different routes
  • This is where Express comes in. Let's compare
    • 
  const express = require('express')
  const app = express();
  
  app.get('/', (req, res) => {
      res.send('hello from the route')
  })
  
  app.get('/api/users', (req, res) => {
      res.send(users)
  })
  
  app.listen(3000, function(){
      console.log('listening on port 3000')
  })
