Fundamentals of the web

  • Web Development: The process of building sites and applications for the Web
  • Front-End Development: The development of client/browser code (HTML, CSS, JS), i.e., what the user interacts with and sees
  • Back-End Development: The development of server-side code that handles such functions as routing, data handling, and databases behind the scenes. Ruby, Python, and Java (+ Node!) are all useful here.

Object Oriented & Functional Programming

  • OOP: focuses on structuring imperative code. It creates models of various concepts in a given problem area, then applies them. These models ("objects") are given functionality ("methods") that enable them to perform various actions; each object can also hold information. The fundamental unit in OOP is the object, which can itself contain further objects, like a Russian Matryoshka doll. The idea is to make it easier for human programmers to visualize the program by tying the program to concrete things, which helps the programmer group things logically. The fundamental thing to remember is that in object-oriented programming, everything is an object and every object has methods that allow the programmer to access its functionality
  • FP: treats the problem area as a sequence of equations ("functions")--essentially, a progression of definitions that lead to the answer. The information begins as a number of axioms and culminates in the answer. (Think of it as a logical argument that ends in what you see on the screen.) Functional programming borrows many concepts from formal mathematics, and, as in mathematics (and unlike imperative programming or OOP), it usually does not save data between steps. The fundamental unit in functional programming is the function, which can perform operations as well as produce further functions

Git and Github

  • Git is a free and open source distributed version control system
  • Github is a web-based hosting service for version control using Git

  • Setting up Git on your computer
    • git config --global user.name "YOUR NAME"

  • Terminal for Mac / Command prompt for Windows users
  • Students experiencing an EACCES error should change directory permissions to install Node and npm without the need to use sudo: $ sudo chown -R $(whoami) /usr/local/lib/node_modules

Pseudocoding: Think like a Programmer

In order to become a great programmer, you have to think like a computer. Essentially, you have to think in code. The first step, is to write down whatever you want your computer to do in plain English.

Exercise

  • Imagine a dot with three different-colored buttons below it. When a user touches one of the buttons, the dot changes to the corresponding color. If the selected color is touched again, the dot turns off

how can this be improved further?

Sample Answer

Conditional: If the red button is tapped (event listener):

  • If the dot is not red, turn the dot red and set the dot color (variable) to red
  • If the dot is red, turn the dot off and set the dot color to initial color
