    • What is a Data Type?

    • A data type is an attribute of data which tells the compiler how the programmer intends to use the data
    • These are usually similar across different languages (string, number, etc.)
    • JavaScript is "loosely-typed"

    JavaScript has seven data types

    1. Numbers
    2. Strings
    3. Booleans
    4. Null
    5. Undefined
    6. Object
    7. Symbols

    Arithmetic Operators for Numbers

    • 1 + 2 : addition
    • 2 - 5 : subtraction
    • 6 * 2 : multiplication
    • 6 / 2 : division
    • 5 % 2 : finds remainders (modulus operator)
    • Math.pow(3,2) : is 3 to the 2nd power
    • Math.sqrt(4) : for √(4)
    • Math.random()*10 : for a number between 1 and 10
    • Math.floor(3.14) : to round down
    • Math.ceiling(3.14) : to round up

    The Array

    An array is a single variable that is used to store different elements in order

    All About Arrays

    • let starks = new Array('robb', 'sansa', 'arya'); : normal way of declaring
    • let avengers = ['ironman', 'hulk', 'thor'] : "array literal" way of declaring

    • avengers[0]; : index starts at 0. This would be 'ironman'
    • avengers.length; : in this case, this is 3
    • let enemy = 'thanos'; : we can use strings like arrays
    • enemy[0]; : this would be the first letter ('t')
    • enemy.length; : this would be the total number of characters

    More About Length

    • let avengers = ['ironman', 'hulk', 'thor'];
    • avengers[10] = 'black panther';
    • avengers.length;

    Array Methods

    Arrays come with plenty of Methods that act as helpers

    • a.toString() : Returns a string with the toString() of each element separated by commas
    • a.pop(): Removes and returns the last item
    • a.push(item) : Adds one or more items to the end
    • a.reverse() : Reverses the array
    • a.shift() : Removes and returns the first item
    • a.unshift([item]) : Prepends items to the start of the array

    Iterating Through Arrays

    The 'For' Loop

    let teams = ['Yankees', 'Mets', 'Red Sox', 'Dodgers'];
for (let i = 0; i < teams.length; i++) {
    console.log(teams[i]);
}
    • let i = 0 : 'i' is our starting point
    • i < teams.length : how long should we loop
    • i++ : how 'i' changes after each instance
    start 'i' from index 2; try i--

    For Each: The Same, but Shorter

    let teams = ['Yankees', 'Mets', 'Red Sox', 'Dodgers'];
  teams.forEach(function(team) {
      console.log(teams[team]);
  })
    • Notice the callback
    • MDN for Reference

    Other Iterator Methods

    • Array.every() : (MDN) returns boolean that tests whether all elements in the array pass the test implemented by the provided function
    • Array.some() : (MDN) same as every, but not for all elements
    • Array.filter() : (MDN) creates a new array with all elements that pass the test implemented by the provided function
    • Array.map() : (MDN) creates a new array with the results of calling a provided function on every element in this array
