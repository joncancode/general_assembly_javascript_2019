What is a Data Type?

  • A data type is an attribute of data which tells the compiler how the programmer intends to use the data
  • These are usually similar across different languages (string, number, etc.)
  • JavaScript is "loosely-typed"

JavaScript has seven data types

  1. Numbers
  2. Strings
  3. Booleans
  4. Null
  5. Undefined
  6. Object
  7. Symbols

Arithmetic Operators for Numbers

  • 1 + 2 : addition
  • 2 - 5 : subtraction
  • 6 * 2 : multiplication
  • 6 / 2 : division
  • 5 % 2 : finds remainders (modulus operator)
  • Math.pow(3,2) : is 3 to the 2nd power
  • Math.sqrt(4) : for √(4)
  • Math.random()*10 : for a number between 1 and 10
  • Math.floor(3.14) : to round down
  • Math.ceiling(3.14) : to round up

    Array Methods

    Arrays come with plenty of Methods that act as helpers

    • a.toString() : Returns a string with the toString() of each element separated by commas
    • a.pop(): Removes and returns the last item
    • a.push(item) : Adds one or more items to the end
    • a.reverse() : Reverses the array
    • a.shift() : Removes and returns the first item
    • a.unshift([item]) : Prepends items to the start of the array
