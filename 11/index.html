Today's Objectives

Feedr Project

  • Familiarize yourself with the API documentation news sources
  • Learn how to parse through API documentation
  • Understand how to successfully retrieve information from APIs
  • Start from scratch or fork and clone your starter code
  • Strategize ways to hide the loader and replace the content of the #main container with that of the API
  • Look up other news sources that might be useful for the project

    • Overview

    We will have plenty of classtime to work on this project

  • The feed reader will pull feeds from our favorite blogs and display them on the browser (remember getElementById)
  • The user will be able to filter between publications through the dropdown on the header menu. Clicking/tapping on one of the articles will load a pop up with more information. The user from that point will be able to either dismiss the additional information or go to the referenced article
  • You'll give the user the ability to pull from 2-3 news sources. You can select the three provided news sources and/or add your own:
  • Mashable, Reddit, Digg are three options
  • This has a few others (NY Times, a movie db, etc), but may be a bit more challenging and need api keys, etc
  • There is starter code here which you may use (or start from scratch)
  • If you decide to use the starter code, clone the repository to your computer by running the following commands
    • 
  git clone https://github.com/joncancode/feedr.git
  cd feedr
  • If you do start from scratch, you must put it on Github (good extra git practice)

    • Suggested Ways to Start

  • Start by adding all the DOM functionality first
  • Map out all of the needed fields/properties from each respective feed
  • Start by doing a console.log of the incoming feeds to confirm you have a successful transaction before you start mapping anything out
  • Use good tools like the JSON View chrome extension
  • Think about ways to best standardize all of your incoming data
  • Test small pieces of functionality frequently, to make sure everything is working
  • Use tools such as Stack Overflow, Google and documentation resources to solve problems

    • More Useful Tips

  • If you want to get the top 10 articles, for example, try pairing map() with a for loop
  • To add stuff to the DOM, remember getElementById, append, and other vanilla JS methods, or use jQuery selectors
  • For styling, Bootstrap may be a good thing to experiment with. Remember to add the CDN
